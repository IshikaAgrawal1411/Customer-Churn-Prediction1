# 🔮 Customer Churn Prediction

## 📌 Problem
Companies lose revenue when customers 
leave undetected. This model predicts 
who will churn BEFORE they leave.

## 📊 Dataset
IBM Telco Customer Churn
7,043 customers | 15 features

## 🔬 What Makes This Project Special
✅ Compared 4 ML Models
✅ Fixed class imbalance using SMOTE  
✅ Evaluated with ROC AUC (industry standard)
✅ Calculated real business ROI

## 🏆 Model Results
| Model | Accuracy | ROC AUC |
|---|---|---|
| Logistic Regression | 76.08% | 0.8248 |
| Random Forest | 77.93% | 0.8298 |
| XGBoost | 78.35% | 0.8370 |
| Gradient Boosting | 78.71% | 0.8480 🏆 |

## 💰 Business Impact
- Correctly identified churners: 283
- Revenue saved: ₹8,49,000
- Retention spend: ₹2,33,000
- Net benefit: ₹6,16,000
- ROI: 264%

## 💡 Key Insight
Customers with HIGH monthly charges + 
SHORT tenure + MONTH TO MONTH contract 
are most likely to churn
